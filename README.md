# 🌿 Plant Disease Prediction using Deep Learning

A deep learning project that automatically identifies plant diseases from leaf images using a **Custom Convolutional Neural Network (CNN)** built with **PyTorch**. This system supports early disease detection and contributes to smart agriculture.

## 🚀 Features

* Custom CNN architecture
* Image preprocessing and data augmentation
* Multi-class disease classification
* Training and validation pipeline
* Built with PyTorch

## 🛠️ Tech Stack

* Python
* PyTorch
* NumPy
* Matplotlib
* Scikit-learn

## 📂 Dataset

This project uses the **PlantVillage Dataset**.

🔗 **Dataset Link:**
https://www.kaggle.com/datasets/abdallahalidev/plantvillage-dataset

## 📊 Model Pipeline

```
Dataset
   ↓
Preprocessing
   ↓
Data Augmentation
   ↓
Custom CNN
   ↓
Training & Validation
   ↓
Prediction
```

## 🎯 Objective

Develop an accurate and efficient deep learning model for automated plant disease detection to assist in crop monitoring and management.

## 📈 Results

The model achieved strong classification performance, demonstrating the effectiveness of CNNs for plant disease recognition.
